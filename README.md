# I Love R - Community Platform

A Shiny-based platform for R programming enthusiasts to share data visualizations, host meetups, discuss AI integration, and stay updated on R news and packages.

## Features

- **Data Visualization**: Interactive charts for manufacturing and other data analysis
- **Meetup Calendar**: Schedule and manage R coding meetups
- **Data Sharing**: Upload and analyze CSV/Excel files
- **News & Packages**: Share latest R updates, package introductions, and AI tips
- **Community Discussion**: Forum for AI and R discussions

## Installation

1. Install R from https://cran.r-project.org/
2. Install required packages:
   ```r
   install.packages(c("shiny", "ggplot2", "dplyr", "bslib", "readxl"))
   ```

## Running the App

```r
library(shiny)
runApp("i_love_r")
```

## Deployment

Deploy to shinyapps.io or other hosting services.
